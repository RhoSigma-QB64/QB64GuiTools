Polygon analysis, filling and alteration

The following functions are based on algorithms and code samples described by Darel Rex Finley under the "Geometry/Math" chapter on his web site http://alienryderflex.com/tutorials.shtml, go there to learn more about it. Note that in my implementation I've limited the polygon corner coordinates to integers, as this library is intended for drawing on screen, not for theoretical exact math.

Table of Contents

POLYGON FUNCTIONS

  • PointInPolygon
    • check whether a point is inside a given polygon
  • PolygonInPolygon
    • check how much two polygons do overlap
  • FillPolygon
    • draw a polygon with given fill- and/or border colors
  • OffsetPolygon
    • calculate a new polygon by shrinking or enlarging a given one

HELPER FUNCTIONS

  • LineIntersection
    • find the intersection point of two lines
