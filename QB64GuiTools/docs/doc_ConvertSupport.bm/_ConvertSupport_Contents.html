Convert point coordinates

Following are some functions to convert the coordinates of a given point between the carthesian and polar system. Also available are functions to move or rotate a point to a new position. Note that the x/y coordinates precision is limited to integers, as the intended use is for converting pixel positions on screen, not for theoretical exact math.

Table of Contents

SYSTEM CONVERSION

  • PolToCartX & PolToCartY
    • convert given polar coordinates to carthesian coordinates
  • CartToPolA & CartToPolR
    • convert given carthesian coordinates to polar coordinates

POINT CONVERSION

  • MovePointX & MovePointY
    • move a point by given distance and direction
  • RotPointX & RotPointY
    • rotate a point by given angle around any given center

 

 

Colorspace functions

Following are functions to convert color values between the RGB and HSB/HSV colorspace and some convenience functions for the HSB/HSV colorspace, which mimic the respective RGB colorspace functions provided by QB64 itself.

Table of Contents

HSB/HSV CONVENIENCE FUNCTIONS

  • HSB32 & HSBA32
    • mimics _RGB32/_RGBA32, but is using HSB/HSV colorspace values to designate the color
  • HUE32 & SAT32 & BRI32
    • mimics _RED32/_GREEN32/_BLUE32, but returns the HSB/HSV colorspace values of the given color

RAW COLOR CONVERSION

  • RGBtoHSB
    • convert a given color from RGB to HSB/HSV values
  • HSBtoRGB
    • convert a given color from HSB/HSV to RGB values
