GetBufLen (function)

This function will return the current amount of data (in bytes) in the specified buffer. The returned size value remains valid until you write or delete data in the respective buffer.

SYNTAX:

size& = GetBufLen& (handle%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, from which you wanna retrieve the current data size.

RESULT:

size& (LONG)

  • The current amount of data (in bytes) present in the buffer, which is limited to a maximum of 1GiB per buffer (see Common-Info).
