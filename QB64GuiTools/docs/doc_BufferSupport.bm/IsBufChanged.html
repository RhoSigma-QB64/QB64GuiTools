IsBufChanged (function)

This function will return a boolean value (flag), indicating whether the current buffer contents was changed or not. New via CreateBuf() created empty buffers are "changed" by default, while those buffers loaded from a file via FileToBuf() will remain "unchanged" until any data is written or deleted. Regularly all writes or deletes will cause a "changed" state. A call to BufToFile() will reset a buffer to "unchanged".

SYNTAX:

change% = IsBufChanged% (handle%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, from which you wanna check the "Change" state.

RESULT:

change% (INTEGER)

  • Is true (-1), if the buffer contents is "changed", false (0) otherwise.
