SetBufFind (function)

This function will define new search criteria. The given criteria will be copied for internal use and a unique ID number for later use with the functions FindBufFwd() and FindBufRev() is returned. You may define up to 100 different criteria in each buffer.

SYNTAX:

findID% = SetBufFind% (handle%, find$)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, for which you wanna define the new search criteria.

find$ (STRING)

  • The search criteria to look for, when calling the actual search functions with the returned ID:
    • This can be a regular text string to look for, but can also contain control chars and converted numbers (_MK$ or STR$), just as you need.
    • For a delimiter search this string should simply list all chars, which shall be recognized as delimiters (eg. ",.{[()]}").

RESULT:

findID% (INTEGER)

  • On success (positive value) this is the ID number of the new defined criteria, on failure (negative value) it is any of the error numbers defined in the simplebuffer.bi file (see Common-Info).

NOTES:

  • You should always remove no longer needed search criteria to make its IDs available for reuse.
