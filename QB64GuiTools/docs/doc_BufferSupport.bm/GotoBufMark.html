GotoBufMark (function)

This will set the current buffer position to the position marked by the bookmark with the given ID.

If you not wanna go to the exact position of the bookmark, but just want to use it as origin for a jump with a certain displacement, then you may use SeekBuf() instead, specifying your desired displacement and give the bookmark ID as the mode parameter.

SYNTAX:

err% = GotoBufMark% (handle%, markID%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, in which you wanna jump to the marked position.

markID% (INTEGER)

  • The ID number of the desired bookmark. You got the ID from the respective SetBufMark() call.

RESULT:

err% (INTEGER)

  • This is usually zero (0) on success, or any of the error numbers defined in the simplebuffer.bi file (see Common-Info), the old buffer position remains unchanged in the error case.
