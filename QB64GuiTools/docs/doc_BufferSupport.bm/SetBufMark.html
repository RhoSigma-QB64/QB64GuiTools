SetBufMark (function)

This function will bookmark the current buffer position. The marked position will automatically be tracked while writing or deleting data in the buffer. You can later go back to the marked position by using the GotoBufMark() function or you may use the bookmark as origin in a SeekBuf() call. You may set up to 100 bookmarks in each buffer.

SYNTAX:

markID% = SetBufMark% (handle%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, in which you wanna set the bookmark.

RESULT:

markID% (INTEGER)

  • This is the ID number of the new set bookmark on success, or the error SBE_NoMoreIDs, if all bookmarks are already in use (see Common-Info).

NOTES:

  • You should always remove no longer needed bookmarks to make its IDs available for reuse.
