RemoveBufMark (function)

This function will remove (unset) the bookmark with the given ID number, it's the counterpart to the SetBufMark() function. The given ID is returned to the buffer system for reuse. Use this function to remove no longer needed bookmarks.

SYNTAX:

err% = RemoveBufMark% (handle%, markID%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, from which you wanna remove the bookmark.

markID% (INTEGER)

  • The ID number of the bookmark to remove. You got the ID from the respective SetBufMark() call.

RESULT:

err% (INTEGER)

  • This is usually zero (0) on success, or error SBE_BadIDNumber, if the given ID number was out of the valid range (see Common-Info).
