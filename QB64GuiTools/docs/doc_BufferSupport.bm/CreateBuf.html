CreateBuf (function)

This function will prepare a new empty buffer for use and return its buffer handle. The buffer will use the OS native line endings mode and its initial state is "changed".

SYNTAX:

handle% = CreateBuf%

RESULT:

handle% (INTEGER)

  • The handle of the new empty buffer, which you must later pass to the various other buffer functions to designate this buffer for operation.
  • If no more buffers are available, then error SBE_NoMoreBuffers is returned (see Common-Info).

NOTES:

  • You may change the line endings mode at any time by calling the appropriate conversion subroutine ConvBufToNativeEol(), ConvBufToWinEol() or ConvBufToLnxMacEol().
