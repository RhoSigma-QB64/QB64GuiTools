CloneBuf (function)

This function will copy the entire contents of the given source buffer into a new buffer and return the new buffer's handle. Also all bookmarks and search criteria are duplicated and the current buffer position, the active line endings mode and the current "change" state of the source buffer are also taken over into the new buffer. Hence, you get an exact clone of the source buffer.

SYNTAX:

handle% = CloneBuf% (handle%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, which you wanna clone.

RESULT:

handle% (INTEGER)

  • The handle of the new cloned buffer, which you must later pass to the various other buffer functions to designate this buffer for operation.
  • If no more buffers are available, then error SBE_NoMoreBuffers is returned (see Common-Info).
