DisposeBuf (sub)

This subroutine will destroy the buffer designated by the given handle and erase all its data. The given handle is returned to the buffer system for reuse. After disposing a buffer it should be obvious, that you can no longer use its handle for any other function calls.

SYNTAX:

DisposeBuf handle%

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, which you wanna dispose.
