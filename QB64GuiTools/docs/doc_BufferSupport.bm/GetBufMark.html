GetBufMark (function)

This function will return the current byte position of the bookmark with the given ID in the specified buffer. The returned position remains valid until you write or delete data in the respective buffer, which could cause any bookmarks to move to different positions.

SYNTAX:

markPos& = GetBufMark& (handle%, markID%)

INPUTS:

handle% (INTEGER)

  • This is the handle of any valid buffer, from which you wanna query the bookmark position.

markID% (INTEGER)

  • The ID number of the desired bookmark. You got the ID from the respective SetBufMark() call.

RESULT:

markPos& (LONG)

  • On success (positive value) this is the retrieved bookmark position, on failure (negative value) it is any of the error numbers defined in the simplebuffer.bi file (see Common-Info).
