Table of Contents

  • Globally SHARED Variables
    • see which variables are globally available and what they contain

  • MessageBox$
    • show a MessageBox with one or more buttons
  • FileSelect$
    • show a File Open or Save dialog box
  • GetUniqueID$
    • get a unique ID string (eg. for temporary file names)
  • TempLog
    • register a temporary file for auto-delete at program end
  • SetPrintMode
    • allows to set a print mode by number (according to _PRINTMODE function)
  • InternalErrHandler
    • activate the Internal Error Handler
  • UserErrHandler
    • activate the User Error Handler
  • Bin$
    • convert any number into a binary string (similar to HEX$ and OCT$)
  • IndexFormat$
    • enhanced PRINT USING style formatting, returns string w/o printing
  • LStrip$ & RStrip$
    • these are more versatile usable LTRIM$/RTRIM$ synonyms
  • RInstr
    • do INSTR from the right side of a string (_INSTRREV synonym)
  • ParseLine
    • break up a line into its individual words or quoted components

  • CurrDrives$
    • returns the letters of currently accessible drives on the computer
  • CurrDIR$
    • returns the current working directory (_CWD$ synonym)
