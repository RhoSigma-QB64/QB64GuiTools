GuiTools base library

This document describes the common base stock of SUBs and FUNCTIONs available for use in all applications made with the GuiTools Framework. You also find a list of the useful shared variables.

Table of Contents

GENERAL NOTES

  • Globally SHARED Variables
    • see which variables are globally available and what they contain

MISC FUNCTIONS

  • InternalErrHandler
    • activate the Internal Error Handler
  • UserErrHandler
    • activate the User Error Handler
  • GetUniqueID$
    • get a unique ID string (eg. for use as temporary file names)
  • MessageBox$
    • show a MessageBox with one or more buttons
  • SetPrintMode
    • allows to set a print mode by number (according to _PRINTMODE function)

STRING FUNCTIONS

  • BinStr$
    • convert any number into a binary string (_BIN$ synonym)
  • IndexFormat$
    • enhanced PRINT USING style formatting with several new features
  • LStrip$ & RStrip$
    • these are more versatile usable LTRIM$/RTRIM$ synonyms
  • RInstr
    • do INSTR from the right side of a string (_INSTRREV synonym)
  • ParseLine
    • break up a string into its individual words or quoted components

FILESYSTEM FUNCTIONS

  • CurrDrives$
    • get the letters of currently available drives on the computer
  • CurrDIR$
    • get the current working directory of your program (_CWD$ synonym)
  • PathPart$
    • get the path out of any given file or folder specification
  • FilePart$
    • get the file/folder name out of a given file or folder specification
  • FileNamePart$
    • get the name part (ie. without extension) of any file or folder name
  • FileExtension$
    • get the extension (ie. without name part) of any file or folder name
  • FileSelect$
    • show a File Open or Save dialog box
  • SearchFile$
    • recursively search a file in a given drive or folder
  • SafeOpenFile
    • try to open a file in an error safe way to avoid program abort
  • TempLog
    • register a temporary file for auto-delete at program end
