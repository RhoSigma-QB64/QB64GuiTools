RInstr (function)

Search a substring in the given source string starting from the end (right side) of the source string. The behavior is equal to the new _INSTRREV function introduced in QB64 v1.3 (incl. Phoenix Edition), but I recommend using RInstr() instead, if you want your program to be usable with the older QB64 versions too.

SYNTAX:

position& = RInstr& (start&, source$, find$)

INPUTS:

start& (LONG)

  • The start position (>0) for the search counted from the left side.
    • Invalid values (<= zero, or > LEN(source$)) will regularly start the search from the end (right side) of the source string, hence passing zero is ok, if you don't need a specific start position.

source$ (STRING)

  • The source string to be searched for an exact match (inclusive the written case) of the given substring (find$).

find$ (STRING)

  • The substring to look for in the given source string.
    • Once again, written case matters for the search (Foo <> foo).

RESULT:

position& (LONG)

  • The position in the source string where the substring (find$) was found, or zero if the search found no matches.
    • Also returns 0 if the source string was empty or shorter than the searched substring (find$).
    • Returns LEN(source$) if the substring (find$) was empty.
