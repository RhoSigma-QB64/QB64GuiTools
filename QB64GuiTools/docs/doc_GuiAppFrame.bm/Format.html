Format$ (function)

Use this function to save a PRINT USING formatted output into a string variable, rather than printing it. This is a multiple name function for the use of upto three arguments.

SYNTAX:

formatted$ = Format$  (fmt$, arg$, typ%)
formatted$ = Format2$ (fmt$, arg$, typ%, arg2$, typ2%)
formatted$ = Format3$ (fmt$, arg$, typ%, arg2$, typ2% arg3$, typ3%)

INPUTS:

fmt$ (STRING)

  • The format template, which may consist of literal text and PRINT USING style format options.

arg$ (STRING)

  • The argument(s) to format according to the fmt$ string, use STR$(num) to pass in a number.

typ% (INTEGER)

  • The type(s) of the respective argument(s),
    • 0 = argument is a real (alphanumeric) string
    • 1 = argument is a STR$ number string

RESULT:

formatted$ (STRING)

  • The PRINT USING formatted string.

NOTES:

  • This function is adapted and slightly altered for multiple arguments use from an idea posted in the QB64 Forum by Fellippe Heitor.
