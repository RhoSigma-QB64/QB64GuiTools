Table of Contents

  • Common Info
    • informations about common inputs and results

  • ModifyBrightness
    • change the general brightness of the given image
  • ModifyContrast
    • change the contrast of the given image
  • ModifyGamma
    • change the graylevel brightness (gamma) of the given image

  • MakeGrayscale
    • make the given image into a B&W photograph (strip color information)
  • MakeAntiqueTint
    • make the given image looking like an old yellowed photograph
  • MakeNegative
    • make the given image into a photograph negative (inverts all colors)

  • ShiftARGB
    • swap (re-order) the component channels of the given image
  • LimitARGB
    • crop each component's intensity of the given image into a defined range

  • AddConstant
    • add a constant (component wise) to each pixel of the given image
  • SubConstant
    • sub a constant (component wise) from each pixel of the given image

  • ExtractChannels
    • extract the specified component channels from the given image
  • ExtractBitfields
    • extract the specified bitplanes (component wise) from the given image

  • ApplyFilter
    • apply the specified filter to the given image (convolution)
